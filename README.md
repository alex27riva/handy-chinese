# Hàndy

**Hàndy** is a pun: **hàn** (汉) + **handy** = useful app for Chinese.

Offline-ready Chinese vocabulary & phrase app for travel in China.
Built as a single-file PWA. Add to Home Screen on iOS for app-like experience.

## Features

- App navigation vocabulary (Alipay, WeChat, maps, etc.)
- Travel phrases (food, transport, emergencies, shopping)
- Tap-to-speak using native iOS/Android TTS
- Fully offline once installed

## Install

Open the site in Safari → Share → Add to Home Screen.
